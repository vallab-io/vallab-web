# Vallab 소개 사이트 프로젝트 계획서

## 1. 프로젝트 목표

Vallab 회사와 팀원을 소개하는 현대적이고 반응형 웹사이트를 구축합니다.

### Vallab 브랜드 정체성

- **회사명**: Vallab (Value + Laboratory의 합성어)
- **핵심 목표**: "단순히 서비스를 만들어드리는 것이 아닌 **가치를 만들겠습니다**"
- **주요 서비스**: Vality - 창작자를 위한 뉴스레터 SaaS 서비스
- **도메인**: vallab.io

## 2. 타겟 사용자

- 잠재 고객/파트너
- 채용 후보자
- 미디어/언론 관계자
- 일반 방문자

## 3. 핵심 요구사항

### 필수 기능
- ✅ Vallab 회사 소개
- ✅ 만들고 있는 주요 서비스 (Vality) 소개
- ✅ 만들어줬던 서비스들 (외주 프로젝트) 포트폴리오
- ✅ 개인 소개 페이지 (About Me)
- ✅ 외부 링크 연결 (소셜 미디어, 프로젝트 링크 등)
- ✅ 반응형 디자인
- ✅ 빠른 로딩 속도
- ✅ SEO 최적화

### 선택 기능
- 다크 모드
- 다국어 지원
- 블로그/뉴스 섹션
- 프로젝트 갤러리

## 4. 페이지별 상세 계획

### 4.1 홈페이지 (/)

**목적**: 첫인상과 주요 정보 제공

**컨텐츠**:
- 히어로 섹션 (Hero Section)
  - Vallab 로고/브랜딩
  - 회사 슬로건: "단순히 서비스를 만들어드리는 것이 아닌 가치를 만들겠습니다"
  - Vallab = Value + Laboratory 설명
  - CTA 버튼 (서비스 보기, 문의하기 등)
- 만들고 있는 주요 서비스 섹션
  - Vality: 창작자를 위한 뉴스레터 SaaS 서비스
  - 서비스 설명 및 주요 기능
  - 가치 제안
  - 서비스 링크 (웹사이트, 데모 등)
- 만들어줬던 서비스들 섹션 (외주 프로젝트)
  - 각 외주 프로젝트 카드 형태로 표시
  - 프로젝트명, 설명, 기술 스택
  - 프로젝트 이미지/스크린샷
  - 프로젝트 링크 (가능한 경우)
  - 클라이언트 정보 (공개 가능한 경우)
- 푸터로 연결

**디자인 요소**:
- 현대적이고 깔끔한 레이아웃
- 강조 색상 사용
- 이미지/일러스트레이션

### 4.2 About Me (/about-me)

**목적**: 개인 소개 및 경력/포트폴리오 링크 제공

**컨텐츠**:
- 프로필
  - 프로필 사진
  - 이름
  - 역할/포지션
  - 간단한 소개
- 경력/경험
  - 현재 역할
  - 주요 경력
  - 교육 배경 (선택사항)
- 기술 스택
  - 주요 기술
  - 기술 아이콘/배지
- 프로젝트/포트폴리오
  - 주요 프로젝트 링크
  - GitHub 링크
  - 블로그 링크
- 소셜 미디어
  - LinkedIn
  - GitHub
  - 블로그/웹사이트
  - 이메일

**섹션 구조**:
1. 프로필 헤더
2. 소개
3. 경력
4. 기술 스택
5. 프로젝트/링크
6. 연락처

### 4.3 Contact (/contact)

**목적**: 연락처 정보 및 문의 폼 제공

**컨텐츠**:
- 연락처 정보
  - 이메일
  - 전화번호
  - 주소
  - 소셜 미디어
- 문의 폼 (선택사항)
  - 이름
  - 이메일
  - 제목
  - 메시지
- 지도 (선택사항)
