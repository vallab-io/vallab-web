**목적**: 첫인상과 브랜드 정체성 전달

**컨텐츠**:
- Vallab 로고

- 서브 헤드라인: Vallab = Value + Laboratory

### 3. 만들어줬던 서비스들 섹션 (외주 프로젝트)

**제목**: "만들어줬던 서비스들" 또는 "Our Works" 또는 "Portfolio"

**레이아웃**: 그리드 또는 카드 형태

**각 프로젝트 카드에 포함할 정보**:
- 프로젝트명
- 프로젝트 설명 (1-2 문장)
- 사용된 기술 스택 (태그 형태)
- 프로젝트 이미지/스크린샷
- 프로젝트 링크 (가능한 경우)
- 클라이언트 정보 (공개 가능한 경우)
- 프로젝트 기간 (선택사항)

**예시 구조**:
```
[프로젝트 카드 1]
- 프로젝트명: [이름]
- 설명: [설명]
- 기술: React, Next.js, TypeScript
- 이미지: [스크린샷]
- 링크: [URL] (있는 경우)

[프로젝트 카드 2]
...
```

## 디자인 고려사항

### 프로젝트 카드 디자인
- 일관된 카드 레이아웃
- 호버 효과 (상세 정보 표시)
- 이미지 최적화 (WebP 형식)
- 반응형 그리드 (모바일: 1열, 태블릿: 2열, 데스크톱: 3열)

### 기술 스택 표시
- 태그/배지 형태
- 색상으로 구분 (선택사항)
- 아이콘 사용 (선택사항)

### 링크 처리
- 외부 링크는 새 탭에서 열기
- `rel="noopener noreferrer"` 추가
- 링크 아이콘 표시 (외부 링크임을 명시)
