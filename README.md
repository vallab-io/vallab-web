# Vallab 소개 사이트

Vallab 회사와 팀원을 소개하는 Next.js 기반 웹사이트입니다.

## 빠른 시작

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 기술 스택

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **UI Components**: shadcn/ui
- **Styling**: Tailwind CSS
- **Deployment**: Vercel
- **Domain**: vallab.io

## 프로젝트 구조

```
web/
├── app/                    # Next.js App Router
├── components/             # React 컴포넌트
├── lib/                    # 유틸리티 함수
├── public/                 # 정적 파일
└── docs/                   # 프로젝트 문서
```
